                ->내장된 Tomcat 서버가 배포를 시작한다
                    (이건 브라우저가 자동으로 열리지 않기 때문에 각자 브라우저에 주소 입력해야한다!(localhost 이런식으로))
                    (옆사람 거 보려면 옆사람 ip입력하면 됨)

프로젝트 구조

  • src/main/java : 자바 소스 코드를 모아두는 폴더
  • src/main/resources : 자바에서 사용할 자원, 설정 파일을 모아두는 폴더
    • templates : Java 템플릿(응답 화면)을 모아두는 폴더(thymeleaf, html)
    • static : templates에 적용될 정적 자원을 모아두는 폴더(JS, CSS, 이미지 파일)
    • application.properties : Spring Boot 프로젝트 관련 설정을 정의하는 설정 파일 (중요!)
  • Project and External Dependencies : build.gradle에 명시된 라이브러리 모음
  • build.gradle : gradle(빌드도구 == maven의 pom.xml)
    프로젝트에 필요한 라이브러리, 배포, 빌드 관련 설정을 작성하는 파일 여기에 작성하면 자동으로 라이브러리를 다운받는다
  1. 요청 + forward 처리
  2. 정적 자원(js,css,image)을 연결하는 방법 연습하기
  3. 서버로 파라미터 제출하기(자바)
