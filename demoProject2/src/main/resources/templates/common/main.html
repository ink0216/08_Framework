2. Thymeleaf

        * 웹  및  독립  실행형  환경  모두를  위한  최신  서버  측 Java 템플릿  엔진
            ->웹 실행 == 요청에 대한 포워드 된 화면
            ->독립 실행 ==jsp와 달리 Thymeleaf(html)는 독립 실행 가능(인터넷 연결X)
                html파일은 인터넷 연결 안돼도 실행된다
        * HTML 파일에서 th(Thymeleaf 전용 속성) 속성을  이용해  
            컨트롤러로  부터  전달  받은데이터를  이용해  동적  페이지를  만들  수  있음. 
            Spring Boot에서는 JSP가  아닌 Thymeleaf 사용을  권장(강제)하고  있음
